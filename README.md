🚀 Excited to share my latest project: **Student Management System**!

As part of my Java Full Stack development journey, I built a Student Management System to strengthen my understanding of frontend-backend integration.

### 🛠️ Tech Stack

* React (Vite)
* Spring Boot
* MySQL
* Axios
* REST APIs

### ✨ Features

✅ Add Student
✅ View All Students
✅ Update Student Details
✅ Delete Student
✅ REST API Integration
✅ MySQL Database Connectivity

### 📚 What I Learned

* Building REST APIs using Spring Boot
* Connecting React with Spring Boot using Axios
* Managing React state with `useState` and `useEffect`
* Working with Props and Component Communication
* CRUD Operations
* Handling asynchronous API calls
* Building a reusable React component structure
